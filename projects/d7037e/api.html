Pythia API documentation version v1

https://pythia.azure-api.net/v1

The API will be versioned so the endpoints will be e.g. recommend/v1/predictions or backoffice/v1/weights.

/recommend/predictions

/recommend/predictions get

get

Get predictions based on the input products

get /recommend/predictions

Get predictions based on the input products

  • Request
  • Response

Query Parameters

  • company: required(string)

    The company name

    Example:

    Lele
  • store_id: required(string)

    The particular store`s ID

    Example:

    3455
  • products: required(array of )

    An array of products ID:s to make the prediction for.

    Example:

    [
  "55345",
  "54344",
  "7337"
]

HTTP status code 200

Returns n predictions as an array of prodcuts ID:s and the model that predicted that product. (FPG = FP-Growth, MAB = Multi Armed Bandit, VIP = Products weighted high in backoffice)

Body

Media type: application/json

Type: any

Example:

{
  "predictions": [
    {
      "product_id": "7194",
      "model": "VIP"
    },
    {
      "product_id": "7193",
      "model": "FPG"
    },
    {
      "product_id": "7192",
      "model": "MAB"
    }
  ]
}
